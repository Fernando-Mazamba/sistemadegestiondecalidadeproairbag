La organización debe establecer, implementar y mantener procesos que aseguren que la producción y prestación de servicios cumplan con los requisitos definidos, garantizando el control adecuado en cada etapa.

Enfoque de Planificación

Para asegurar que la operación se ejecute de manera eficaz, la planificación debe considerar:

1 - Determinación de Requisitos

- Identificación de requisitos del cliente, normativos y legales.

- Consideración de necesidades de las partes interesadas.

2 - Definición de Procesos
- Establecimiento de las etapas operativas necesarias.
- Identificación de entradas, actividades y salidas de cada proceso.
- Asignación de responsabilidades y recursos.

3 - Criterios para la Ejecución de la Operación
- Definición de controles para garantizar la calidad del producto o servicio.
- Aplicación de métodos de seguimiento y medición para verificar conformidad.

4 - Gestión de Riesgos y Oportunidades
- Evaluación de riesgos que puedan afectar la calidad.
- Implementación de acciones para mitigar riesgos y aprovechar oportunidades.

5 - Evidencia Documentada
- Mantenimiento de registros para demostrar conformidad con los requisitos.
- Implementación del Control Operacional

Para asegurar que la operación sea controlada de manera efectiva, la organización debe:

Establecer procedimientos y métodos documentados para la producción y prestación del servicio.
Garantizar la disponibilidad de recursos como personal capacitado, equipos adecuados y materiales.
Aplicar criterios de control y monitoreo para verificar la calidad y prevenir no conformidades.
Realizar evaluaciones periódicas para identificar oportunidades de mejora.
