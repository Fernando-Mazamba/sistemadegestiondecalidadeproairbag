La organización debe asegurarse de que los productos y servicios que no cumplen con los requisitos establecidos sean identificados y controlados para evitar su uso o entrega no intencionada.

8.7.1 Identificación y Control de No Conformidades

Cuando se detecta una no conformidad en un producto o servicio, la organización debe:

  1. Identificar y segregar el producto no conforme para evitar su uso inadecuado.
  2. Registrar la no conformidad, incluyendo la naturaleza del problema y las acciones tomadas.
  3. Evaluar el impacto de la no conformidad en la calidad y en el cliente.
  4. Determinar y aplicar las acciones correctivas necesarias.

Las opciones para gestionar un producto o servicio no conforme incluyen:

  • Corrección: Reprocesar o rectificar el producto para que cumpla con los requisitos.
  • Aprobación bajo concesión: En casos justificados, obtener la autorización del cliente o de una autoridad competente para aceptar el producto con desviaciones.
  • Degradación o uso alternativo: Reasignar el producto para otro uso donde la no conformidad no afecte la calidad o seguridad.
  • Desecho o eliminación: Si el producto no puede corregirse, debe eliminarse de forma controlada para evitar riesgos.

8.7.2 Control Posterior a la Entrega

Si se detecta una no conformidad después de la entrega del producto o prestación del servicio, la organización debe:
Informar al cliente sobre el problema detectado y las acciones a tomar.

Aplicar las medidas correctivas necesarias, como devoluciones, reparaciones o reemplazos.
Investigar la causa raíz para evitar la recurrencia del problema.
Actualizar los registros y procedimientos para reflejar las lecciones aprendidas.

Conclusión
El control de productos y servicios no conformes es fundamental para garantizar la calidad y la satisfacción del cliente. Implementar una gestión eficaz de las no conformidades permite reducir riesgos, mejorar la eficiencia y fortalecer la confianza en la organización.
